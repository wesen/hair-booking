Integration notes

  • The browser initiates login through /auth/login.
  • The server exchanges the authorization code at /auth/callback.
  • A signed HTTP-only session cookie carries the authenticated identity.
  • The React app uses fetch("/api/me", { credentials: "include" }) to read that session.
